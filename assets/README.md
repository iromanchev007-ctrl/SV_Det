# Папка assets

Сюда можно положить изображения сайта.

## Фон главного экрана

Чтобы на главном экране появилось ваше фото автомобиля (как на макете),
положите файл с именем:

```
assets/hero.jpg
```

Он подхватится автоматически. Рекомендации:
- тёмное фото на чёрном фоне, ширина от 1920px;
- формат `.jpg` или `.webp` (для `.webp` поменяйте путь в `css/styles.css`,
  переменная `--hero-image`);
- вес желательно до ~400 КБ для быстрой загрузки.

Если файла нет — на главном экране показывается аккуратный тёмный градиент
с золотым бликом, сайт работает без ошибок.

## Логотип

`assets/logo.png` — используется в микроразметке для поисковиков (необязательно).
