# СВ Детейлинг — сайт с онлайн-записью

Современный адаптивный одностраничный сайт детейлинг-студии в Пензе.
В основе — **система онлайн-записи**: клиент оставляет марку, модель и пожелания,
и заявка готова к отправке менеджеру.

Чёрно-белый стиль с золотыми акцентами, плавные анимации появления блоков,
hover-эффекты, SEO-разметка и быстрая загрузка. Без сборщиков и зависимостей —
чистые HTML, CSS и JavaScript.

## Структура проекта

```
.
├── index.html        # разметка страницы
├── css/styles.css    # стили и адаптив
├── js/script.js      # меню, анимации, марки/модели авто, форма
├── assets/           # сюда кладётся фон hero.jpg (см. assets/README.md)
├── vercel.json       # конфиг для Vercel
├── netlify.toml      # конфиг для Netlify
├── robots.txt
└── sitemap.xml
```

## Возможности

- **Онлайн-запись.** Форма с именем, телефоном, email, выбором услуги,
  **маркой и моделью авто** (выпадающие списки + ручной ввод) и комментарием.
  После отправки — сообщение «Спасибо! Мы свяжемся с вами в ближайшее время».
- **Марка → модель.** Модели подставляются по выбранной марке; если нужной нет —
  можно ввести вручную.
- Кнопки «Записаться онлайн» и «Позвонить», блоки «О компании», «Преимущества»,
  «Услуги и цены», «Адрес» с Яндекс.Картой, «Связь» и подвал.
- Адаптив для ПК и смартфонов, бургер-меню, маска телефона.

## Локальный запуск

Это статический сайт — достаточно открыть `index.html` в браузере.
Либо поднять локальный сервер:

```bash
python3 -m http.server 5173
# затем открыть http://localhost:5173
```

## Публикация

### Vercel
1. Загрузите проект в репозиторий GitHub.
2. На [vercel.com](https://vercel.com) → **Add New → Project** → импортируйте репозиторий.
3. Framework Preset: **Other**, build-команда не нужна. Нажмите **Deploy**.

### Netlify
1. Загрузите проект в репозиторий GitHub.
2. На [netlify.com](https://netlify.com) → **Add new site → Import an existing project**.
3. Build command оставьте пустым, **Publish directory** = `.`. Нажмите **Deploy**.

> Можно и без GitHub: в Netlify просто перетащите папку проекта в окно загрузки.

## Подключить отправку заявок (когда понадобится)

Сейчас отправка — заглушка (данные пишутся в консоль браузера).
Чтобы заявки реально приходили менеджеру, в `js/script.js` в обработчике
`form submit` отправьте `payload` на ваш бэкенд / Telegram-бота / CRM / Formspree.
